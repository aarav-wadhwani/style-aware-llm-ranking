Style-Aware LLM Ranking

Bradley-Terry Model Statistical Inference Bias Correction

Evaluating Large Language Models (LLMs) is complicated by human biases. This report introduces a method to decouple content quality from superficial presentation.

Abstract

We present a rigorous statistical framework for ranking LLMs using the Bradley‚ÄìTerry model augmented with style covariates. Our analysis of a large-scale pairwise battle dataset reveals that standard win-rate metrics are heavily confounded by response length. By controlling for verbosity and formatting, we derive a "style-free" ranking that better reflects intrinsic model reasoning capabilities.

1. The Data Landscape

Reliable ranking requires understanding the underlying data distribution. We analyzed thousands of pairwise comparisons ("battles") to identify coverage gaps and imbalances.

2. Statistical Methodology

We employ the Bradley‚ÄìTerry model, which posits that the odds of Model \( i \) beating Model \( j \) depend on their latent strength difference.

The Model Formulation

The probability of a win is modeled via the logistic function:

$$ P(i \succ j) = \frac{1}{1 + e^{-(\theta_i - \theta_j)}} $$

To correct for bias, we extend this to the Style-Adjusted Model, introducing a vector of style features \( S_{ij} \) (e.g., length difference, formatting usage):

$$ P(i \succ j) = \sigma(\underbrace{\theta_i - \theta_j}_{\text{Skill}} + \underbrace{\beta^T S_{ij}}_{\text{Style Bias}}) $$

The Incentive Problem

This correlation suggests that models can "game" the leaderboard simply by generating longer, fluffier answers. This hurts efficiency and user experience in real-world applications.

When we mathematically control for length (and other features like Markdown usage), the ranking shifts. This "purified" ranking rewards concise, high-quality reasoning.
